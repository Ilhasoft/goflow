Context

The context is all the variables which are accessible in expressions:

Channel

  • __default__ the name of the channel (text)
  • uuid the UUID of the channel (text)
  • name the name of the channel (text)
  • address the address of the channel (text)

Contact

  • __default__ the name or URN of the contact (text)
  • uuid the UUID of the contact (text)
  • id the numeric ID of the contact (text)
  • first_name the first name of the contact (text)
  • name the name of the contact (text)
  • language the language of the contact as 3 (text)
  • created_on the creation date of the contact (datetime)
  • urns the URNs belonging to the contact (text)
  • groups the groups the contact belongs to (group)
  • fields the custom field values of the contact (fields)
  • channel the preferred channel of the contact (channel)

Group

  • uuid the UUID of the group (text)
  • name the name of the group (text)

Result

  • __default__ the value of the result (text)
  • name the name of the result (text)
  • value the value of the result (text)
  • category the category of the result (text)
  • category_localized the localized category of the result (text)
  • input the input of the result (text)
  • extra the extra data of the result such as a webhook response (any)
  • node_uuid the UUIF of the node in the flow that generated the result (text)
  • created_on the creation date of the result (datetime)

Root

  • contact the current contact (contact)
  • fields the current contact custom fields values (fields)
  • urns the current contact URN values (urns)
  • results the current run results (results)
